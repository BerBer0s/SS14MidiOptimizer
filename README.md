# MIDI Оптимизатор для Space Station 14

Инструмент с графическим интерфейсом для подготовки MIDI файлов к использованию в игре Space Station 14.

## Что он делает

- **Чинит зависшие и дублирующиеся ноты** — предотвращает вечно звучащие звуки в SS14
- **Нормализует velocity** — отдельные лимиты для ударных и мелодических дорожек
- **Обрезает тишину в начале** — файл начинается сразу с первой ноты
- **Группирует треки по каналам** — структура SS14-совместимого MIDI
- **Поддерживает повреждённые файлы** — восстанавливает битые заголовки и байты
- **Drag & Drop** — перетащи папку или файлы прямо в окно

## Скачать

👉 [Последний релиз (.exe)](../../releases/latest)

Скачай `MIDI_Optimizer_SS14.exe` — дополнительная установка не нужна.

## Использование

1. Запусти `MIDI_Optimizer_SS14.exe`
2. Перетащи папку с MIDI файлами или выбери через кнопку «Выбрать»
3. Настрой параметры если нужно
4. Нажми «▶ Обработать ВСЕ»
5. Готовые файлы появятся рядом с исходниками с суффиксом `_ss14.mid`

## Параметры

| Параметр | По умолчанию | Описание |
|---|---|---|
| Velocity ударных | 85 | Лимит громкости для канала 10 (GM drums). 85 — стандарт SS14 |
| Velocity мелодии | 127 | Лимит громкости мелодических дорожек |
| Авто-подтверждение | выкл | Обрабатывать повреждённые файлы без вопроса |

## Сборка из исходников

Требования: Python 3.11, pip

```bat
cd папка с файлами
build.bat
```

EXE появится в `tobuild/dist/MIDI_Optimizer_SS14.exe`
